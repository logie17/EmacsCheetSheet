# Vim 2 Emacs

I'm a slow learner and it takes me about x^n times before something is concrete in my mind.


* Searching

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | /foo          | Ctrl-s foo    |

* Undo

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | u             | Ctrl-xu, Ctrl-/  |

* Abort?

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | esc           | Ctrl-g        |

* Window Hopping

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | ctrl-ww       | Ctrl-xo       |

* Buffer loading

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | :b{n}, :ls    | Ctrl-xb buffer_name, Ctrl-x-Ctrl-b (see buffers ) |

* Quitting

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | :q    | Ctrl-x-Ctrl-c |

* Saving

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | :w    | Ctrl-x-Ctrl-s |

* Help

    | Vim           | Emacs         |
    | ------------- | ------------- |
    | :help command    | F1-command |
